Class MessageBox

lucee.Component
    cbmessagebox.MessageBox

Copyright Since 2005 Ortus Solutions, Corp www.ortussolutions.com --- This module provides a way to track alert message boxes. The user has several types of message types 1. Warn 2. Error 3. Info The messages and optional metadata will be stored in the application's Flash RAM storage. --- The look and feel of the messages can be altered by styles and cfml template settings.
